## **Système Multi-Tenant**

URACAD utilise un système multi-tenant qui permet à plusieurs organisations d'utiliser la même application tout en gardant leurs données complètement séparées. C'est comme un immeuble où chaque organisation a son propre appartement privé.

## **Fonctionnement**

### **Structure**

- **Organisations** : Espaces privés indépendants
- **Utilisateurs** : Peuvent appartenir à plusieurs organisations
- **Données** : Isolées et sécurisées par organisation
- **Rôles** : Personnalisables par organisation

### **Avantages**

- **Pour les Organisations**
  - Données 100% privées
  - Personnalisation complète
  - Gestion indépendante
  - Coûts d'infrastructure partagés
- **Pour les Utilisateurs**
  - Une seule application
  - Accès multi-organisations
  - Rôles personnalisés
  - Interface familière
