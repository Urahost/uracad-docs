## Vue d'ensemble

Cette API constitue un pont de communication essentiel entre votre Mobile Data Terminal (MDT) et les serveurs FiveM utilisant les frameworks ESX ou QB-Core. Elle a été conçue pour automatiser et simplifier l'accès aux données des citoyens stockées dans la base de données de votre serveur de jeu de rôle.

## Objectif principal

L'objectif principal de cette API est de permettre la **récupération automatique et en temps réel des informations des citoyens** directement depuis la base de données du serveur FiveM. Cette intégration élimine le besoin de saisie manuelle des données et garantit que les informations affichées dans le MDT sont toujours à jour et synchronisées avec l'état actuel du serveur.

## Fonctionnalités clés

- **Récupération automatique des profils citoyens** : Accès instantané aux données personnelles, antécédents, et informations de base
- **Synchronisation en temps réel** : Les modifications apportées sur le serveur sont immédiatement reflétées dans le MDT
- **Compatibilité multi-framework** : Support natif pour les serveurs ESX et QB-Core
- **Interface unifiée** : Une seule API pour gérer les données, quel que soit le framework utilisé

## Avantages

Cette solution automatisée permet aux forces de l'ordre et aux équipes administratives de :

- Accéder rapidement aux informations des citoyens sans quitter l'interface MDT
- Réduire les erreurs liées à la saisie manuelle
- Améliorer l'efficacité des contrôles et des enquêtes
- Maintenir une cohérence des données entre le serveur de jeu et les outils externes

L'API est conçue pour être robuste, sécurisée et facilement intégrable dans votre écosystème existant.
