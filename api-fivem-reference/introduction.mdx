# API QBCore/ESX Citizens

Cette API REST open source permet de récupérer les informations des citoyens et leurs véhicules depuis les bases de données des frameworks **QBCore** et **ESX** de FiveM.

## Projet Open Source

Cette API est un projet **open source** que vous pouvez déployer sur votre propre serveur. Chaque communauté FiveM peut installer et configurer l'API selon ses besoins spécifiques.

 \<CardGroup cols={2}\> \<Card title="Installation" icon="download" href="/fivem-api/install"\> Guide d'installation de l'API \</Card\> \<Card title="Configuration" icon="gear" href="/fivem-api/configuration"\> Configuration personnalisée \</Card\> \</CardGroup\> 

## Vue d'ensemble

L'API offre des endpoints pour accéder aux données des citoyens de manière programmatique, que ce soit pour des applications externes, des sites web, ou des outils d'administration.
